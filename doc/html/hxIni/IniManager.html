class hxIni.IniManager

Available on all platforms

Primary INI manager class. Used for reading and writing INI files. For data format, checkout http://en.wikipedia.org/wiki/INI_file Also, check out http://stackoverflow.com/questions/3870019/javascript-parser-for-a-string-which-contains-ini-data @author Wikiti
